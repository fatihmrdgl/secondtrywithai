# MimCRM Project Overview

MimCRM is a comprehensive customer relationship management platform designed to streamline business interactions and improve customer service. This project consists of a backend built with ASP.NET Core 8 and a frontend developed using React.js.

## Features
- User Management
- Customer Interaction Tracking
- Reporting and Analytics

## Technologies
- Backend: ASP.NET Core 8
- Frontend: React.js with Vite/CRA
